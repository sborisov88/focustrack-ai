# Сравнение AI-инструментов

Этот файл оставлен как стабильная точка входа для сдачи ДЗ 1: в формулировке задания ожидается сравнительная таблица и обоснование выбора инструмента.

Основная таблица находится в [`comparison_table.md`](comparison_table.md). В ней зафиксированы:

- минимум 4 инструмента, включая cloud-агентов и IDE/CLI-агентов;
- критерии: доступ к контексту проекта, скорость генерации, качество кода, стоимость, безопасность данных, удобство интеграции;
- балльная оценка по шкале 1-5;
- обоснование по каждому инструменту;
- актуальные источники;
- итоговый выбор Codex как основного AI coding assistant для FocusTrack AI.

Итоговый отчёт по ДЗ 1 находится в [`report.md`](report.md).
