# Сравнительная таблица AI-инструментов для FocusTrack AI

Дата актуализации: 17.06.2026.

Оценка по шкале от 1 (слабо подходит) до 5 (отлично подходит).

Критерии сравнения: доступ к контексту проекта, скорость генерации, качество кода, стоимость, безопасность данных, удобство интеграции.

Главная развилка в выборе между Codex и Cursor — не функциональность, а стоимость владения. По основным инженерным возможностям оба инструмента находятся на одном уровне: локальный репозиторий, git, терминал, многофайловые правки, агентный режим и работа с контекстом проекта. Codex получает преимущество потому, что уже включён в подписку ChatGPT, а Cursor требует отдельного тарифного плана.

## Отдельно: LLM-модели для будущего AI-слоя

GigaCode, YandexGPT, DeepSeek и Qwen важны для выбора модели и провайдера, но основной объект сравнения здесь — **AI coding assistant** как рабочий инструмент разработки. Для FocusTrack AI они рассматриваются как возможный LLM-слой:

| Модель / направление | Роль в проекте | Вывод |
|----------------------|----------------|----------------|
| GigaCode / GigaChat | Российская экосистема, возможна для корпоративных сценариев | Не основной coding-agent интерфейс для разработки проекта |
| YandexGPT | Российский LLM-провайдер | Можно учитывать для будущего AI-слоя, но не заменяет IDE/CLI-агента |
| DeepSeek | Open-source / доступные модели для reasoning и code | Кандидат для вызовов через OpenRouter после отдельной оценки качества |
| Qwen | Open-source семейство моделей | Кандидат для on-premise или OpenRouter-сценариев, не основной инструмент разработки |

Итог: для **разработки проекта** выбран Codex; для **продуктового AI-слоя** модели будут оцениваться отдельно при реализации backend/Edge Functions.

## Общий вывод по таблице

Cloud-агенты выигрывают по скорости первого экрана, но проигрывают по контексту, безопасности и долгосрочной работе с локальным репозиторием.

Cursor остаётся сильной IDE-альтернативой и по ключевым инженерным возможностям находится на одном уровне с Codex. Поэтому итоговый выбор строится вокруг стоимости: **Codex выбран как основной AI coding assistant для FocusTrack AI**, потому что он уже включён в подписку ChatGPT и не требует отдельной оплаты за дополнительную IDE-среду.
