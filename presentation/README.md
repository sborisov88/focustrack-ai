# FocusTrack AI — структура защиты

## 1. Проблема

Человеку сложно удерживать фокус на длинной личной или рабочей цели (полумарафон, IELTS, финансовая подушка, запуск пет-проекта): уточнение цели, план задач, прогресс, weekly review и личные заметки живут в разных местах.

## 2. Решение

FocusTrack AI — AI-планировщик личных и рабочих целей: объединяет цель, AI-уточнение, AI-план задач, прогресс, еженедельное AI-ревью и ответы по личным заметкам (RAG) в одном рабочем интерфейсе.

## 3. Демо-сценарий (на жизненной цели)

1. Открыть дашборд: «Категории целей» и «Личный планировщик».
2. Создать цель «Пробежать первый полумарафон».
3. Пройти AI-уточнение и получить AI-план задач.
4. Отметить задачу выполненной и показать пересчет прогресса.
5. Запустить еженедельное AI-ревью (weekly review).
6. Задать RAG-вопрос по личным заметкам: «на какой неделе была самая длинная пробежка».
7. Показать Supabase schema, RLS и Edge Functions.
8. Показать Playwright screenshots/video.

## 4. Архитектура

```text
React + shadcn
  -> Supabase client
  -> PostgreSQL + RLS
  -> Edge Functions
  -> OpenRouter
```
